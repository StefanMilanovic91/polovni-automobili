# Polovni Automobili - Local setup

### Requirements: PHP + XAMPP

1. Use `git clone https://github.com/StefanMilanovic91/polovni-automobili.git` or `Download ZIP`
2. Open `XAMPP` and `Start` MySQL
3. Open the project with your code editor
4. From the code editor, open Command Prompt terminal
5. Execute the seed file - `mysql -u root -p < seed.sql`
6. Run `php -S localhost:8888 -t public`
7. Visit http://localhost:8888/
